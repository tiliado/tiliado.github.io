Install Nuvola Player 3.0 from Terminal

Nuvola Player 3.0 has reached the end-of-life status on December 31st, 2017. Check out Nuvola Apps 4 instead.

Not recommended to less experienced users

This installation method is not recommended to less experienced users. Any mistake can break your system.

Steps

  1. Follow instructions listed at Add Repository → Manual Way. For Nuvola Player 3.0 Stable Releases, choose the stable component codename.
  2. Check format support: Ubuntu, Debian, Fedora.
  3. Explore features: Unity, GNOME or terminal.

Questions? Feedback?

Having trouble? We'd like to help! Search for information in the archives of the Nuvola Player Users mailing list, or post a question.
