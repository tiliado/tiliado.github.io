Installation Instructions

Nuvola Project provides cross-distribution flatpak packages of Nuvola 3.1, which contain all requires dependencies. In order to install a particular Nuvola app, follow steps in the Nuvola Apps Repository Index portal according to your distribution:

  • Ubuntu 16.04 Xenial Xerus
  • Ubuntu 16.10 Yakkety Yak
  • elementaryOS 0.4 Loki
  • Debian 8 Jessie (stable)
  • Debian 9 Stretch (testing)
  • Debian Sid (unstable)
  • Fedora 25
  • Fedora 24
  • Arch Linux
  • Other linux
