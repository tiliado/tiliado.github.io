In Development - Feedback Welcome

Nuvola Player 3 is a new code-base written from scratch and the current development stage can be described as alpha, so please be patient and ask questions and provide feedback whenever you find a bug or two.

Development Roadmap

You can check more detailed roadmap at Github.

  • October 2014 - Alpha builds
  • November 2014 - Beta builds
  • December 2014 - Stable release

Support Development

As development of Nuvola Player is not funded by any company, foundation nor organization, it completely depends on donations from users, Tiliado Patrons. Without them Nuvola Player would be a history.

There are many awesome things to to. Make a donation to support development

Info

Nuvola Player 3 Alpha won't replace Nuvola Player 2 in your system. You can keep using Nuvola Player 2 until Nuvola Player 3 is stable enough for you.
